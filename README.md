# ContiGoMobile / Math Rush: Desafio da Esfinge

Repositório do projeto Unity (jogo mobile).

## Dependência local: Layer Lab — GUI Pro - Casual Game

O projeto utiliza o pacote comercial **Layer Lab** listado na Unity Asset Store com o nome **exato**:

| Campo | Valor |
|--------|--------|
| **Fornecedor** | Layer Lab (LAYERLAB) |
| **Nome exato (Asset Store)** | **GUI Pro - Casual Game** |
| **Página do produto** | https://assetstore.unity.com/packages/2d/gui/gui-pro-casual-game-176695 |
| **Versão de referência (Asset Store)** | **4.1.2** *(última versão indicada na página do produto em 2026-05-06; confirme em *My Assets* / Package Manager após importar)* |
| **Caminho local típico após import** | `Assets/Layer Lab/GUI Pro-CasualGame/` *(nome da pasta pode coincidir com o pacote Unity; ajuste se o import usar outro caminho)* |

Esta pasta **não é versionada** neste repositório (está em `.gitignore`). Cada desenvolvedor ou máquina de CI deve **importar o asset comprado** localmente (Unity Package ou `.unitypackage` conforme a loja).

Documentação do fornecedor (guia): [Layer Lab User Asset Guide](https://tinyurl.com/LayerLabAssetGuide)

## Requisitos

- Unity (versão do projeto: ver `ProjectSettings/ProjectVersion.txt`).
- Conta/licença válida para o asset Layer Lab acima, instalada localmente.
